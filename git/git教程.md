git 教程
==========================================
## **git分支**
```md
git branch                   --显示所有分支
git branch   <分支名字>       --创建分支
git checkout <分支名字>       --检查分支
```
------------------------------------------
## **本地仓库**
git status                  --检查本地仓库的变化情况
git diff                    --查看更改内容
